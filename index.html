How to Access

1. Get a valid license key by subscribing

2. Send a GET request to the API 

curl https://honey-api.niceland.tech:44444/ip/[LICENSE]/[TIME_AFTER_UNIX]

TIME_AFTER_UNIX (unix timestamp): Used for filtering. The API will only send you IPs which have been see AFTER the given timestamp 

Response: [ {IP:"0.0.0.0", Hit:0, Count:0} , ...]

Count (int): Tells us how many packets this IP has sent to the honeypot

Hit (unix timestamp): Last time the honeypot saw this IP
